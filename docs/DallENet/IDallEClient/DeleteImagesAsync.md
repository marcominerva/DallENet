# IDallEClient.DeleteImagesAsync method

Triggers the deletion of generated images.

```csharp
public Task DeleteImagesAsync(string operationId, CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| operationId | The GUID that identifies the original image generation request. |
| cancellationToken | The token to monitor for cancellation requests. |

## Return Value

The Task corresponding to the asynchronous operation.

## Exceptions

| exception | condition |
| --- | --- |
| [DallEException](../../DallENet.Exceptions/DallEException.md) | An error occurred while calling the API. |

## Remarks

Generated images are automatically deleted after 24 hours by default, but this method can be used to explicitly trigger the deletion earlier. This method deletes all the images that have been generated in the request associated with the given *operationId*.
